# 项目介绍
还为收集到太多名片而无处存放感到苦恼？还在为互换名片时而找不到名片尴尬？
我想基于人工智能的名片识别小程序能帮到你。 

本次「云开发」公开课，将通过实战“名片小程序”开发，
教你如何借助小程序·云开发能力，提升功能开发效率，在商务礼仪中更能出色的展现自己。
